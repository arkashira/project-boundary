# Scope Health Dashboard
A real-time scope health visibility dashboard for project maintainers.

## Features
* Displays key metrics related to scope health
* Metrics include completion percentage, deviation from original scope, and pending approvals
* Updates in real-time as changes occur

## Usage
1. Create a `ScopeHealthDashboard` instance
2. Update metrics using `update_metrics` method
3. Get metrics using `get_metrics` or `get_all_metrics` methods
4. Convert metrics to JSON using `to_json` method
