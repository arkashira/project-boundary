# Partner Targets
## Overview
As project-boundary aims to help open source project maintainers and developers define and manage project scope, we need to identify potential partners that can help us achieve this goal. The following list outlines specific SaaS/APIs to integrate with, along with their free-tier limits, integration effort, and value-add.

## Partner Integration Roadmap

### Priority 1: Affiliate/Revenue-Share Partners

1. **GitHub**
	* Free-tier limit: 5000 API requests per hour
	* Integration effort: M
	* Value-add: Automate project scope management for GitHub projects, providing a seamless experience for users.
	* Affiliate/revenue-share: Yes, GitHub offers an affiliate program for developers.
2. **GitLab**
	* Free-tier limit: 5000 API requests per hour
	* Integration effort: M
	* Value-add: Integrate project-boundary with GitLab's project management features, enhancing the user experience.
	* Affiliate/revenue-share: Yes, GitLab offers an affiliate program for developers.
3. **Bitbucket**
	* Free-tier limit: 5000 API requests per hour
	* Integration effort: M
	* Value-add: Provide a seamless experience for Bitbucket users by automating project scope management.
	* Affiliate/revenue-share: Yes, Bitbucket offers an affiliate program for developers.
